# AI-Vtuber 功能说明

## 核心卖点

- 多模型容灾回复：DeepSeek / GLM / OpenAI 自动切换 + 本地语料兜底
- 低延迟语音播报：异步合成 + 顺序播放 + 队列背压
- 日语直播实时字幕：音频捕获、ASR、中文翻译、OBS 叠加层一体化
- 可配置化部署：路径、设备、模型、并发、超时全部可调

## 能力清单

### 1) 直播弹幕 AI 互动

- 实时接收弹幕并生成主播回复
- 支持用户级上下文记忆
- 支持欢迎词、冷场心跳、昵称纠正

### 2) 语音合成与播放

- Edge TTS 与 Vocu 双后端
- `mpv` 播放与设备控制
- 失败重试和超时保护

### 3) 实时字幕与翻译

- 支持 URL 直拉音频与系统回采
- faster-whisper 持久化识别
- 异步翻译队列（失败冷却、重试、统计）
- 双语/原文/译文三路输出

### 4) OBS Overlay

- 本地 HTTP 服务
- API 拉取字幕 JSON
- 浏览器源可直接挂载
- 响应缓存优化，降低磁盘读取频率

### 5) 本地训练能力

- `db.txt` 一问一答训练
- 写入 `db.sqlite3` 供主程序兜底问答

## 适用客户

- 虚拟主播个人作者
- 小型直播工作室
- 多语种内容搬运与直播团队
- AI 陪伴直播产品原型团队
